# 🏨 Revenue Insights in Hospitality Domain – Power BI Project
## 🔎 Project Overview

Revenue Insights in the Hospitality Domain is a Power BI analytics solution designed to analyze hotel performance across revenue, occupancy, pricing, and booking trends.

This project simulates a real-world hotel chain scenario where management requires dynamic dashboards to monitor KPIs such as Revenue, ADR, RevPAR, Occupancy %, and Week-over-Week (WoW) performance.

The solution is built using structured data modeling, DAX measures, and hospitality-specific business metrics to enable data-driven decision-making.

## 🎯 Business Objectives

- Monitor overall revenue performance across hotels and cities

- Analyze Occupancy %, ADR, and RevPAR trends

- Track Week-on-Week (WoW) growth across KPIs

- Evaluate booking behavior and realization rates

- Identify high-performing cities and properties

- Support pricing and capacity optimization decisions

## 🏗️ Data Architecture

- **The project is built using:**

## 📊 Key Metrics Explained

- Revenue – Total booking revenue generated

- Occupancy % – DURN / DSRN

- ADR – Revenue per utilized room

- RevPAR – Revenue per available room

- DSRN – Daily room capacity available for sale

- DBRN – Daily rooms booked

- DURN – Daily rooms actually utilized

- WoW % – Weekly growth comparison of KPIs
